# Internal Ops Dashboard

A full-stack internal support request tracker built with Node.js, Express, and SQLite. This project demonstrates a complete CRUD workflow, CSV export, and a simple frontend for internal application support.

## Features

- **CRUD API**: Create, Read, Update, Delete requests via REST endpoints
- **Database**: SQLite relational database with structured tables
- **Data Export**: Export requests as CSV
- **Frontend**: Simple dashboard to add and view requests
- **JSON Responses**: Structured data for internal systems

## Tech Stack

- Backend: Node.js, Express
- Database: SQLite
- Frontend: HTML, CSS, Vanilla JavaScript
- Tools: Postman (API testing), Git/GitHub (version control)
